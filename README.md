# Solidity Playground

This is a simple visual representation of how the frontend working with smart contracts.


##  💡 Usage

### Requirements

All you need is any Web3 provider like Metamask.


## ⚙️ Quick start

### 💿 Install

```bash
yarn
```

### 💻 Run dev server

```bash
yarn dev
```

### 🔨 Build project

```bash
yarn build
```
